# dodgyca

A very quick and dirty python script for running a development CA.

## Install

Install python and openssl.

## Usage

./dodgeca.py -ca hello

./dodgeca.py -cert hello

### Options

[-h] halp

[-ca] create a new ca

[-cert] create a new cert

name name of ca
